# Simple JavaScript OnClick Project

This is a basic web project that demonstrates a simple interaction using HTML, CSS, and JavaScript. When a button is clicked, the text content of a paragraph is changed.

## Files

- **index.html:** The main HTML file that contains the structure of the web page.
- **styles.css:** The CSS file for styling the elements on the page.
- **script.js:** The JavaScript file that handles the interaction and changes the text content.
